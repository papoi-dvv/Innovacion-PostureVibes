# PostureVibes

Dispositivo IoT para corrección de postura corporal con retroalimentación háptica. Documentación del proceso de desarrollo completo desde la conceptualización hasta el prototipo final.

## Descripción del Proyecto

PostureVibes es un dispositivo wearable que monitorea la postura del usuario y proporciona retroalimentación mediante vibración cuando detecta una postura incorrecta. El proyecto incluye:

- **Hardware**: Sensor de inclinación, microcontrolador ESP32, motor vibrador y batería recargable
- **Carcasa**: Diseño 3D ergonómico impreso en PLA
- **Documentación**: Proceso completo de desarrollo semanal

## Tecnologías Utilizadas

- **Frontend**: React + TypeScript + Vite
- **Routing**: React Router DOM
- **Styling**: CSS modules y estilos inline
- **Build Tool**: Vite con HMR

## Instalación y Uso

```bash
# Clonar el repositorio
git clone <repository-url>
cd posturevibes

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Construir para producción
npm run build
```
